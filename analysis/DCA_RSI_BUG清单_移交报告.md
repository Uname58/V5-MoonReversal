## 一、策略是什么

每周一 UTC 0:00 定投 BTC，买入金额由 RSI 决定倍数，卖出由「浮盈 15% + RSI≥70」触发。

```
买入:  RSI < 20 → 3x ($60)
       RSI < 25 → 2x ($40)
       其他     → 1x ($20)

卖出:  总仓位浮盈 ≥ 15% 且 RSI ≥ 70 → 卖出 30% 仓位
       (无冷却，每根 4h K 线独立判断)
```

---

## 二、已确认的 BUG（4 个）

### BUG 1 · 连续绞肉式清仓 · 严重

**位置**：`simulate_dca_rsi()` 卖出逻辑（L222-237）

```python
for i in range(len(df)):          # 每根 4h K 线都执行
    if profit >= 0.15 and rsi >= 70:
        sell_qty = position * 0.3  # 卖出当前仓位的 30%
```

**问题**：没有冷却期。牛市中 RSI≥70 可连续几十根 K 线（几天到一周），每次触发卖 30%。仓位指数级衰减：

```
第 1 根：100% → 卖 30% → 剩 70%
第 2 根：70%  → 卖 30% → 剩 49%
第 3 根：49%  → 卖 30% → 剩 34%
第 5 根：剩 17%
第 10 根：剩 2.8%（归零）
```

**2020 年实测**：BTC +295%，DCA+RSI 只赚 +15%。仓位在 $10k→$29k 主升段被清空。

### BUG 2 · 买入速度 = 本金的 0.2%/周 · 严重

每次定投 $20，本金 $1,280。满仓需要 50 周。止损清仓后，重建仓位需一年。

**2023/2024 实测**：新版 B+D 虽修了绞肉机，但止损触发后到年底仓位未恢复，依然跑输 Plain DCA 25-34%。

### BUG 3 · RSI 分级买入几乎不触发 · 中等

- RSI < 20 (3x)：六年触发 < 10 次（BTC 极少跌到 RSI 20）
- RSI < 25 (2x)：每年 2-5 次，多投 $20-40
- 99% 的时间是 1x，行为 ≈ Plain DCA

**结论**：RSI 买入模块六年贡献 < $2,200（总本金 $10,000），年化 < 3%。

### BUG 4 · 亏损集中在牛市 · 结构性

| 年份 | BTC 涨幅 | B&H | 旧版 D+R |
|------|----------|-----|----------|
| 2020 | +295% | +295% | +15% |
| 2023 | +157% | +152% | +15% |
| 2024 | +124% | +112% | +12% |

牛市交出 90-95% 涨幅。熊市 (2022) 只减亏到 -14%（vs B&H -65%）。

牛熊不对称：熊市少亏 50%，牛市少赚 90% → 长期必然跑输。

## 四、已尝试的修复：B+D 版

**文件**：`dca_trailing_backtest.py`

**改动**：删除「浮盈 15%+RSI≥70 卖 30%」，改为「持仓期最高点回撤 15% 全仓卖出」。

### B+D 效果

| 指标 | 旧版 | 新版 |
|------|------|------|
| 六年累计 | $14,004 | $16,238 (+16%) |
| 最大熊市亏损 | -13.88% | -5.40% |
| 卖出次数/年 | 43-96（绞肉） | 3-15（正常） |
| 牛市跑输 B&H | 交出 93% | 交出 87% |

**结论**：修了一个 bug，消灭了绞肉机。但六年依然跑输 Plain DCA 63%，瓶颈不在卖出逻辑。

## 五、真正的瓶颈

四个 bug 修完，策略的天花板是 **每周买入 $20**。

- 满仓需 50 周
- 止损后重建一年
- 牛市空仓/轻仓的代价远超任何参数优化能弥补

DCA 变体在 BTC 上的六年上限 ≈ Plain DCA 的 $44,041。加任何东西（RSI、止损）都是减速器，不是加速器。

## 六、改进方向（供接手 AI 参考）

### 不做的事
- 调 RSI 阈值（过拟合）
- 调止盈止损百分比（边际收益接近零）
- 加更多指标到现有 DCA 框架（天花板不变）

### 建议做的事

1. **仓位再平衡**：止损后不是每周 $20 重建，而是一次性回到目标仓位%（如 60%）
2. **核心/卫星分离**：60% 仓位永远不卖（吃 Beta），40% 做择时
3. **趋势过滤器**：牛市不卖/少卖，熊市允许卖出。一个简单的 MA 金叉/死叉就够了
4. **动态仓位**：波动率低 → 仓位高，波动率高 → 仓位低（ATR-based）
5. **多时间框架**：4h + 日线确认，避免单一 K 线级别的假信号
